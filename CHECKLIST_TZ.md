## ❌ НЕ РЕАЛИЗОВАНО (требуется доработка)

### 1. Экспорт шаблона Excel для создания групп товаров
**ТЗ:** "Экспорт шаблона Excel для удобного создания групп товаров"
**Статус:** ✅ РЕАЛИЗОВАНО
**Реализовано:** 
- Кнопка "Скачать шаблон Excel" в админке групп товаров
- Шаблон содержит примеры для недели (7 листов)
- Каждый лист = меню на день

### 2. Корректировка рабочего календаря
**ТЗ:** "Корректировка рабочего календаря (отпуск, отгул, больничный) для правильного начисления средств"
**Статус:** ✅ РЕАЛИЗОВАНО
**Реализовано:** 
- Модель WorkDayCalendar для отслеживания рабочих дней каждого сотрудника
- Типы дней: рабочий день, отпуск, больничный, отгул, праздничный день
- Задача начисления баланса учитывает календарь сотрудника
- Управление календарем в админке

### 3. История операций с балансом (BalanceTransaction)
**ТЗ:** "Просмотр истории операций по балансу для каждого сотрудника"
**Статус:** ✅ РЕАЛИЗОВАНО
**Реализовано:** 
- Модель BalanceTransaction с типами операций (начисление, списание, корректировка, возврат)
- Просмотр истории в админке
- Автоматическое создание записей при начислении и списании

### 4. Резервирование средств вместо немедленного списания
**ТЗ:** "Блокировка средств (логически вычитается из доступного баланса)" → "Списание: После закрытия группы средства окончательно списываются"
**Статус:** ✅ РЕАЛИЗОВАНО
**Реализовано:** 
- Добавлен статус заказа "reserved" (зарезервирован)
- При создании заказа средства резервируются (статус 'reserved')
- При закрытии группы автоматически списываются средства и меняется статус на 'confirmed'
- Метод `close_selection_and_finalize_orders()` в MenuItemGroup

### 5. Автоматическое формирование итоговых заказов после закрытия группы
**ТЗ:** "Автоматическое формирование итоговых заказов после закрытия выбора для группы товаров"
**Статус:** ✅ РЕАЛИЗОВАНО
**Реализовано:** 
- При закрытии группы автоматически финализируются все заказы (списание средств)
- Автоматическое создание транзакций баланса
- Экспорт отчета доступен в админке

### 6. Экспорт отчета в Excel (сейчас только CSV)
**ТЗ:** "Формирование единого отчета (Excel/CSV)"
**Статус:** ✅ РЕАЛИЗОВАНО
**Реализовано:** 
- Экспорт отчета поддерживает как Excel, так и CSV формат
- Экспорт шаблона Excel для создания групп товаров

### 7. Привязка заказа к группе товаров
**ТЗ:** "Заказы (Order): ID, дата, сотрудник, группа товаров, список позиций, сумма, статус"
**Статус:** ✅ РЕАЛИЗОВАНО
**Реализовано:** 
- Добавлено поле group в модель Order (ForeignKey к MenuItemGroup)
- При создании заказа автоматически определяется группа товаров
- Отображение группы в админке заказов

### 8. Просмотр истории заказов в админке для сотрудника
**ТЗ:** "Просмотр истории заказов и операций по балансу для каждого сотрудника"
**Статус:** ✅ РЕАЛИЗОВАНО
**Реализовано:** 
- В карточке сотрудника добавлены вкладки с историей операций баланса (BalanceTransactionInline)
- В карточке сотрудника добавлены вкладки с историей заказов (OrderInline)
- Удобный просмотр всей истории в одном месте
