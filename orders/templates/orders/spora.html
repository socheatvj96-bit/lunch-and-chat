Руководство по использованию системы заказа обедов

1. Управление сотрудниками

  • Подтверждение аккаунтов: В разделе «Сотрудники» админ-панели необходимо установить флаг «Учётная запись подтверждена», чтобы пользователь мог совершать заказы.
  • Баланс: Начисление (по умолчанию 150₽) происходит автоматически в рабочие дни согласно Глобальному календарю.
  • Лимиты: При достижении «Минимального баланса» (настраивается индивидуально) возможность заказа блокируется.

2. Управление меню

  • Группы товаров: Меню разбито на группы (напр. «Обед на понедельник»). Чтобы группа была видна, она должна быть активна и дата должна попадать в период видимости.
  • Импорт из Excel: Используйте кнопку «Импорт групп товаров» в админке. Система сама распределит блюда по колонкам «Наименование» и «Цена».
  • Медиа-файлы: Для массовой загрузки фото используйте импорт через ZIP-архив с CSV файлом описания.

🖼️ Инструкция по загрузке картинок

  • Подготовка: Соберите все изображения в одну папку. Названия файлов должны быть на латинице (например, pasta.jpg).
  • CSV-файл: Создайте файл import.csv с колонками: name (название блюда в системе) и image (имя файла картинки).
  • Архивация: Выделите все картинки и CSV-файл, упакуйте их в .ZIP архив (не в папку, а именно файлы в корень архива).
  • Загрузка: В админке перейдите в «Импорт блюд с фото», выберите ваш ZIP-архив и нажмите «Загрузить».

3. Процесс заказа

  • Каналы заказа: Доступны через Telegram-бот (/menu) и Web-интерфейс (MiniApp).
  • Резервирование: При создании заказа средства сначала «резервируются» (статус reserved).
  • Списание: Окончательное списание и смена статуса происходит при массовом подтверждении админом или закрытии выбора группы.
  • Отмена заказа: Пользователь может сам отменить заказ в разделе «Мои заказы», пока он находится в статусе «Зарезервирован». Средства вернутся на баланс автоматически.

4. Баланс и пополнение

  • Просмотр реквизитов: В приложении рядом с балансом появилась кнопка +. При нажатии открывается окно с актуальными реквизитами для перевода.
  • Настройка реквизитов: Текст реквизитов правится в админке, раздел «Настройки», ключ payment_details.

5. Служебные функции

  • Бэкап данных: Доступен по скрытому адресу /backup-db/ (только для суперпользователей). Выгружает актуальное состояние базы в формате JSON.
  • Отчетность: Полная выгрузка заказов для бухгалтерии доступна в разделе «Заказы» админ-панели (Excel/CSV).
