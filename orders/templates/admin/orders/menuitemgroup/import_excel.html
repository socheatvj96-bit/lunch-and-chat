Инструкция по импорту

  1. Подготовьте Excel файл (.xlsx) с меню
  2. Каждый лист в файле будет обработан как отдельная группа товаров (меню на день)
  3. Название листа может содержать дату (будет автоматически распознана)
  4. Укажите названия столбцов в вашем Excel файле
  5. Загрузите файл

Формат Excel файла:

Обязательные столбцы:

  • Название - название товара

Необязательные столбцы:

  • Дата - дата для группы (если не указана, будет взята из названия листа)
  • Цена - цена товара (число, можно с запятой или точкой)
  • Описание - описание товара

Пример структуры:

Файл nextweek_menu.xlsx содержит листы:

  • Лист "01.12.2024" - меню на 1 декабря 2024
  • Лист "02.12.2024" - меню на 2 декабря 2024
  • и т.д.

Каждый лист содержит таблицу с товарами на этот день.
