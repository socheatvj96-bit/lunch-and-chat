Инструкция по импорту

  1. Подготовьте CSV файл с данными о товарах
  2. Укажите названия столбцов в вашем CSV файле
  3. Выберите кодировку и разделитель
  4. Загрузите файл

Формат CSV файла:

Обязательные столбцы:

  • Название - название товара
  • Группа - категория товара (будет создана автоматически, если не существует)
  • Ресторан - название ресторана (будет создан автоматически, если не существует)

Необязательные столбцы:

  • Цена - цена товара (число, можно с запятой или точкой)
  • Описание - описание товара
  • Изображение - путь к изображению в ZIP архиве (например: photos_today/img1.png)

ZIP архив с изображениями:

Вы можете загрузить ZIP архив с фотографиями товаров. В CSV файле укажите путь к изображению относительно корня архива.

Пример:

  • ZIP архив: photos_today.zip
  • В архиве: photos_today/img1.png, photos_today/img2.jpg
  • В CSV укажите: photos_today/img1.png

Пример CSV файла:

название;группа;ресторан;цена;описание;изображение
Борщ;Первые блюда;Столовая №1;150;Традиционный украинский борщ;photos_today/borscht.jpg
Салат Цезарь;Салаты;Столовая №1;200,50;Салат с курицей и соусом Цезарь;photos_today/caesar.png
Пицца Маргарита;Пицца;Пиццерия;350;Классическая пицца с томатами и моцареллой;photos_today/pizza.jpg
