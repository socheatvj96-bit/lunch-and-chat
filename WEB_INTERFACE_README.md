# Обновление системы заказа обедов - Веб-интерфейс и импорт CSV

  - ✅ Важные поля (цена, изображения) - подсвечиваются красным мигающим

### 2. Импорт товаров из CSV в Django Admin

#### Доступ к функции импорта
1. Войдите в Django Admin: `https://victor.kiselev.lol/admin/`
2. Перейдите в раздел "Блюда" (MenuItem)
3. Нажмите кнопку "Импорт из CSV" в правом верхнем углу

#### Формат CSV файла

**Обязательные столбцы:**
- `название` - название товара
- `группа` - категория товара (создается автоматически, если не существует)
- `ресторан` - название ресторана (создается автоматически, если не существует)

**Необязательные столбцы:**
- `цена` - цена товара (число, можно с запятой или точкой)
- `описание` - описание товара

#### Пример CSV файла (с разделителем точка с запятой):

```csv
название;группа;ресторан;цена;описание
Борщ;Первые блюда;Столовая №1;150;Традиционный украинский борщ
Салат Цезарь;Салаты;Столовая №1;200,50;Салат с курицей и соусом Цезарь
Пицца Маргарита;Пицца;Пиццерия;350;Классическая пицца с томатами и моцареллой
Суп-лапша;Первые блюда;Столовая №2;120;Куриный суп с домашней лапшой
Греческий салат;Салаты;Столовая №2;180;Свежие овощи с сыром фета
```

#### Настройки импорта

При импорте вы можете указать:
1. **Названия столбцов** - укажите, как называются столбцы в вашем CSV файле
2. **Кодировка** - выберите кодировку файла (UTF-8, Windows-1251, CP1251)
3. **Разделитель** - выберите разделитель (запятая, точка с запятой, табуляция)

#### Процесс импорта

1. Подготовьте CSV файл
2. Откройте страницу импорта в Django Admin
3. Укажите названия столбцов (по умолчанию: название, группа, ресторан, цена, описание)
4. Выберите кодировку и разделитель
5. Загрузите файл
6. Система покажет результаты импорта:
   - Количество созданных товаров
   - Количество обновленных товаров
   - Список ошибок (если есть)

### 3. Новые модели данных

#### MenuItemGroup (Группа товаров)
- Название группы
- Описание
- Порядок сортировки
- Активность

#### MenuItemImage (Изображение товара)
- Связь с товаром
- Файл изображения
- Флаг "Основное изображение"
- Порядок отображения

#### Обновленная модель MenuItem
- Добавлено поле `group` (группа товара) - **обязательное**
- Поле `price` теперь необязательное
- Добавлено поле `updated_at`
- Поддержка множественных изображений

### 4. Увеличенные лимиты загрузки

- Максимальный размер файла: **10 MB**
- Поддерживаемые форматы изображений: JPG, PNG, WebP
- Можно загружать несколько изображений одновременно

## Использование

### Для администраторов

1. **Создание групп товаров**:
   - Зайдите в Django Admin → Группы товаров
   - Создайте категории (например: "Первые блюда", "Салаты", "Десерты")

2. **Добавление товаров через веб-интерфейс**:
   - Перейдите на `https://victor.kiselev.lol/admin/products/`
   - Нажмите "Добавить товар"
   - Заполните обязательные поля (название, группа)
   - Добавьте цену и описание (рекомендуется)
   - Загрузите изображения (рекомендуется)
   - Нажмите "Создать"

3. **Массовый импорт из CSV**:
   - Подготовьте CSV файл с товарами
   - Зайдите в Django Admin → Блюда → Импорт из CSV
   - Настройте маппинг столбцов
   - Загрузите файл

### Для пользователей

1. Откройте `https://victor.kiselev.lol/`
2. Просматривайте доступные товары по категориям
3. В будущем: добавление в корзину и оформление заказа

```python
MEDIA_URL = '/media/'

DATA_UPLOAD_MAX_MEMORY_SIZE = 10485760  # 10MB
FILE_UPLOAD_MAX_MEMORY_SIZE = 10485760  # 10MB
```

### Docker volumes

Убедитесь, что в `docker-compose.yml` есть volume для медиафайлов:

```yaml
volumes:
  - ./media:/app/media
```

## Решение проблем

### Изображения не загружаются

1. Проверьте права доступа к директории media:
   ```bash
   docker-compose exec web chmod 755 /app/media
   ```

2. Убедитесь, что volume смонтирован в docker-compose.yml

### Ошибка при импорте CSV

1. Проверьте кодировку файла (должна быть UTF-8 или Windows-1251)
2. Убедитесь, что разделитель указан правильно
3. Проверьте, что обязательные столбцы присутствуют
4. Проверьте формат цены (число с точкой или запятой)

### CSRF ошибка

Убедитесь, что в `settings.py` добавлен ваш домен:

```python
CSRF_TRUSTED_ORIGINS = [
    'https://victor.kiselev.lol',
]
```
